# 🎯 마이웨이 (My Way)
> 대학생 AI 취준 비서 플랫폼
> **"정보를 보여주는 게 아니라, 판단하고 행동한다"**

---

## 💡 프로젝트 핵심

마이웨이의 진짜 차별점은 **선배 롤모델 기반 취준 갭 분석**이다.

1. **선배 스펙 데이터 축적** — 같은 학과/선배들의 들은 수업, 학점, 복수전공, 자격증, 그리고 **어느 회사·직무에 취업했는지**.
2. **목표 기반 자동 매칭** — 사용자가 가고 싶은 회사/직무를 정하면, AI가 그 목표에 취업한 선배들을 찾아 보여준다.
3. **롤모델 선택** — 사용자가 마음에 드는 선배 한 명을 고른다.
4. **AI 갭 분석 + 액션 플랜** — 내 스펙 vs 선배 스펙을 비교해 "들어야 할 수업 / 따야 할 자격증 / 학점·어학 목표 / 지금 할 일"을 제시한다.

> 기존 취준 서비스는 정보를 *나열*하지만, 마이웨이는 실제 선배 데이터에 근거해 **무엇을 해야 하는지 판단**해준다.

## ✅ 지금까지 한 것 (진행 상황)

### 1. 디자인 — 다크 블루 코스믹 테마 전면 적용
- 앱 전체를 라이트 그린 → **다크 블루 코스믹 테마**로 재설계 (전역 CSS, 사이드바, 입력창, 버튼, 메트릭, 테이블 등).
- **로그인 화면** = 인트로 랜딩 페이지
  - 상단: `myWay` 브랜드 **히어로** — 별·길(도로) SVG가 5갈래로 갈라지는 애니메이션, 가지 끝에 붙은 아이콘(취업·스펙·로드맵·비용·스케줄). (`static/hero.html`, JS 포함)
  - 하단: **아바타 소개 섹션** — 움직이는 캐릭터 + 기능 카드 4개 + 통계. (`static/about.html`)
  - 사이드바 숨김 + 풀스크린 클린 레이아웃.

### 2. 홈 = 개인 대시보드
- 인사말 → **대학생활 진행률 대시보드** → 목표 메트릭 → 빠른 시작 → 배너.

### 3. 대학생활 진행률 (홈에 통합)
- 학점 이수율(%) + 진행 바, 현재/목표 학점, **종강/방학 D-day**, 학기 진행률 바, 이번 학기 수강 과목.
- 데이터 있으면 대시보드, 없으면 입력 폼을 홈에서 바로 처리 (`views/progress.py`).

### 4. ⭐ 선배 매칭 (핵심 기능 — 프로토타입)
- 가짜(seed) 선배 9명 데이터 (`utils/seniors.py`).
- 목표 직무·회사로 선배 자동 매칭 → 선배 카드 목록 → 롤모델 선택 → **AI가 내 스펙 vs 선배 스펙 비교표 + 갭 분석 액션 플랜** 출력 (`views/mentor.py`, `compare_with_senior()`).

### 5. 기타 AI 기능 (기존)
- 스펙 분석, 맞춤 로드맵, 공부 스케줄, 채용공고 탐색(현재 샘플 데이터), 로그인/회원가입/세션 저장.

## 📁 프로젝트 구조
```
myway/
├── app.py                  # 메인 앱 (라우팅, 로그인, 홈, 전역 테마)
├── static/
│   ├── hero.html           # 로그인 상단 길(도로) 히어로 애니메이션
│   └── about.html          # 소개 섹션 (아바타 + 기능 카드)
├── views/
│   ├── onboarding.py       # 스펙 입력
│   ├── analysis.py         # 스펙 분석 (AI)
│   ├── mentor.py           # ⭐ 선배 매칭 (핵심)
│   ├── progress.py         # 대학생활 진행률 (홈에 통합)
│   ├── roadmap.py          # 맞춤 로드맵 (AI)
│   ├── schedule.py         # 공부 스케줄 (AI)
│   ├── cost.py             # 비용 계산기
│   └── crawling.py         # 채용공고 탐색 (현재 샘플 데이터)
└── utils/
    ├── claude_api.py       # AI 연동 (현재 Groq Llama 3.3 사용)
    ├── seniors.py          # ⭐ 선배 seed 데이터 + 매칭 로직
    ├── crawler.py          # 채용공고 (샘플)
    └── session.py          # 로그인/세션 저장·불러오기
```

## 🛠️ 기술 스택
- **Python**
- **Streamlit** — 프론트엔드 / 앱 프레임워크
- **Groq (Llama 3.3 70B)** — AI 엔진 (`utils/claude_api.py`, 파일명과 달리 현재 Groq 사용)
- **HTML/CSS/JS** — `components.html`로 임베드한 애니메이션 히어로/소개 섹션
- **BeautifulSoup / requests** — 채용공고 크롤링용 (현재 미연동)

## ⚠️ 현재 한계 & 다음 단계
- **선배 데이터가 가짜(seed)** — 흐름·UI 데모용. 실제 가치를 내려면 데이터 수집 필요(설문 / 사용자 기여 / 외부 데이터).
- **스펙 입력에 "들은 수업 / 복수전공" 항목 없음** — 선배와 정밀 비교하려면 추가 권장.
- **채용공고가 샘플 데이터** — 실제 채용 API / 크롤링 연동 필요.
- **🔐 보안: Groq API 키가 소스에 하드코딩** (`utils/claude_api.py`) — 즉시 폐기·재발급 후 환경변수/`st.secrets`로 분리 필요.
- (선택) 선배 매칭 결과를 홈 대시보드에도 요약 표시.
